# Student Task Tracker

A C# console application for tracking student tasks in training programs.

## Features
- Task management system
- Fast lookup using Dictionary
- Unique students using HashSet
- FIFO task processing using Queue
- LINQ queries for overdue tasks
- File storage using JSON
- Multithreading task completion

## Technologies
- C#
- .NET
- LINQ
- JSON File Handling
- Multithreading
